# Portfolio Website

A personal portfolio website for Gobishankar G, a Python Full Stack Developer. This project features a "handcrafted" aesthetic with paper textures, soft shadows, and subtle micro-interactions, built using HTML5, Bootstrap 5, and vanilla JavaScript.

## Features

- **Handcrafted Design**: Custom CSS variables for a consistent ivory, teal, and terracotta color palette with paper-like textures.
- **Responsive Layout**: Fully responsive design using Bootstrap 5's grid system.
- **Smooth Navigation**: Smooth scrolling for anchor links and a collapsible mobile navbar.
- **Scroll Animations**: Elements fade in and move up as you scroll using `IntersectionObserver`.
- **Project Modal**: A dynamic modal to display project details without leaving the page.
- **Contact Form Simulation**: A functional-looking contact form with validation and simulated submission feedback.
